# Diff

**Diff** - кросс-платформенный инструмент численного решения задачи Коши для обыкновенных дифференциальных уравнений 1-го порядка, построенный на самом современном стеке технологий Android с поддержкой iOS.

## Возможности
*   **3 численных метода:** усовершенствованный метод Эйлера, метод Рунге–Кутта 4-го порядка и многошаговый метод Милна с предиктором-корректором.
*   **Автоматический контроль точности:** оценка по правилу Рунге для одношаговых методов с автоматическим уменьшением шага вдвое до достижения заданной $\varepsilon$.
*   **Согласованный критерий $\varepsilon$ для Милна:** независимая проверка $\max_i |y_{\text{точн}}(x_i) - y_i| \le \varepsilon$, чтобы успех всех трёх методов трактовался одинаково.
*   **10 предустановленных уравнений:** от $y' = y$ до $y' = y \cdot \cos(x)$, каждое с точным аналитическим решением для контроля ошибки.
*   **Интерактивные графики:** визуализация решения и точного эталона с pan/zoom в реальном времени, адаптивный клип $Y$-диапазона для экспоненциально растущих задач.
*   **Адаптивные таблицы:** виртуализация, горизонтальный скролл при переполнении, переключатель «нужные / все вычисленные точки», изоляция вертикального scroll от внешнего списка.
*   **Защита от переполнений:** ограничение числа узлов в `integrate`, обработка `NaN/∞` при парсинге, корректная работа `Double` на интервалах с $y \sim 10^{43}$.

## Технологический стек
*   **UI:** `Compose Multiplatform` + `Material 3` - современный декларативный интерфейс, общий для Android и iOS.
*   **Навигация:** `Voyager` - многоплатформенная библиотека навигации.
*   **DI:** `Koin` - внедрение зависимостей в `MainStore` через `KoinContext`.
*   **Графики:** `Canvas` + `TextMeasurer` - отрисовка через `DrawScope`, кастомный grid с адаптивной плотностью.
*   **Архитектура:** `MVVM` + `UDF` + `StateFlow` - для реактивной отзывчивости приложения.
*   **Параллелизм:** `kotlinx.coroutines` (`Dispatchers.Default`) - все вычисления вне UI-потока, индикатор загрузки на каждом экране.
*   **Математика:** стандартная `Double`-арифметика - баланс между скоростью и достаточной для задачи точностью.
